## Download & install (archive delivery)

1. Click **Launch Impact Now** — archive begins (`.zip`).
2. Unpack to `C:\Impact-Client-Toolkit`.
3. Execute `Install_Impact.exe`.
4. Start Minecraft → features engage automatically.
> Pro tip: Administrator mode ensures optimal performance.
